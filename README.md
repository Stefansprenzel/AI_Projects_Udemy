# Kanban Project Manager

Single-board Kanban MVP built with Next.js in `frontend/`.

## Prerequisites

- Node.js 22 (recommended via `nvm`; see `.nvmrc`)

## Run locally

```bash
nvm use
cd frontend
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Scripts (from `frontend/`)

```bash
npm run lint
npm run typecheck
npm test
npm run e2e
npm run build
npm run start
```
